To Read Your Files: 
input_deck = Frost.Read("Your Directory")

You need to add this utility code to your code to make it work.

To Retrieve a Key Value (for example): 
value = input_deck.melting_points["titanium"]

Seems you have a file melting_points.xxx in your directory.
